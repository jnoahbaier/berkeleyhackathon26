A small uppercase mono status pill for state and metadata — "Lit", "Night 12", "Locked", "Betrayed".

```jsx
<Badge tone="lit" dot>Lit</Badge>
<Badge tone="brand">Chapter 04</Badge>
<Badge tone="danger" dot>Betrayed</Badge>
```

Tones: `neutral | brand | lit (candle) | success | danger | night`. Set `dot` for a leading status dot.
