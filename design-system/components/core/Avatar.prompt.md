A guild member's avatar, coloured by seat (1–4 → clay rose / sage teal / periwinkle / old gold). Shows initials or an image.

```jsx
<Avatar name="Mara Vance" seat={2} />
<Avatar name="Theo" seat={3} size={56} ring />
```

`seat` drives the colour; `ring` adds a parchment ring + seat halo for overlap on busy backgrounds.
